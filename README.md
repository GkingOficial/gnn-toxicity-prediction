# QSAR GNN

Este repositório contém os códigos desenvolvidos para o **TCC**.

## Pré-requisitos

- [Miniconda](https://docs.conda.io/en/latest/miniconda.html) ou [Anaconda](https://www.anaconda.com/products/distribution) instalados.
- Git instalado (opcionalmente).

## Instalação do ambiente

1. Clone o repositório:

   ```bash
   git clone https://github.com/GkingOficial/TCC.git
   cd TCC
   ```

2. Crie o ambiente Conda a partir do arquivo `environment.yml`:

   ```bash
   conda env create -f environment.yml
   ```

3. Ative o ambiente:

   ```bash
   conda activate env_gnn
   ```

4. Instale as dependências específicas do projeto:

   ```bash
   pip install -r "QSAR GNN/requirements.txt"
   ```
